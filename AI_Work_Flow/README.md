## 12. Tokenization (Most Important Topic)

### What is a Token?

A **token** is a small piece of text converted into a number.

### Token Can Be:

* Word
* Part of word
* Symbol

### Example:

Sentence:

```

```

Tokenized as:

```
[I] [am] [learning] [AI]
```

Converted into numbers:

```
I â†’ 12
am â†’ 45

AI â†’ 999
```

ðŸ‘‰ Computer works only on numbers
